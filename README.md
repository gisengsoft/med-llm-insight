# Med LLM Insights

Academic dashboard for curated medical benchmark analysis with LLMs.

## About the project

**Med LLM Insights** is a web dashboard created to present the results of an academic project on the evaluation of Large Language Models (LLMs) in the medical domain.

The application organizes and visualizes:
- open medical questions and their gold-standard answers
- multiple-choice medical questions
- model responses
- evaluation scores across multiple criteria
- curated project files and reports

This project was developed as a visual and interactive layer on top of the academic materials produced for **Atividade Avaliativa 1**, focused on the medical domain.

## Main features

- Overview dashboard with key metrics
- Open Questions section with:
  - question text
  - gold answer
  - must-have and nice-to-have elements
  - model answers
  - observations
  - evaluation scores and comments
- Notable Cases section highlighting relevant examples
- Model Comparison section comparing:
  - GPT-5.4 Thinking
  - Claude 4.6 Sonnet
  - Gemini 3.0
- Multiple Choice section with curated items and explanations
- Files & Reports section with direct access to project artifacts
